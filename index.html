Инструкция

Страница наглядно показывает алгоритм сортировки бинарным деревом.
Для начала демонстрации нажмите кнопку "Начать следующую сортировку". После этого данная кнопка станет заблокирована. Кнопка разблокируется, когда массив полностью отсортируется. Тогда и будет возможность запустить новую сортировку.
Чтобы остановить сортировку немедленно - нажмите на кнопку "Остановить сортировку" - страница перезагрузится и у вас будет возможность начать сортировку заново.

После нажатия кнопки "Начать следующую сортировку" страница создаст массив для сортировки и отправит его в сортировочную зону, где будет происходить графическая сортировка.

Для создания страницы использовались библиотеки jQuery и SVG.js, а также дополнительная надстройка svg.filter.js для управления фильтрами в SVG, например, такими, как размытие по Гауссу.
